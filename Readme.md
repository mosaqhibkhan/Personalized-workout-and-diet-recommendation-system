# AI Workout & Diet Recommendation System

An AI-powered fitness recommendation web application that generates personalized workout and diet plans based on user body metrics, fitness goals, and experience level.


---

# Tech Stack

## Frontend
- HTML
- CSS
- JavaScript

## Backend
- Node.js
- Express.js

## Machine Learning
- Python
- Flask
- Scikit-learn
